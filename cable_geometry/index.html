    Review points
      Standing assumptions
      • Within group cable spacing is assumed touching unless separately modelled.
      • Mixed service visual uses one worst case OD for all shown services and is schematic only.
      • Bend visual shows a single cable body sweep only and does not model full multi cable sweep, trench profile, duct entry, pull path or installation forces.
      • Burial depth is recorded as an indicative input only.
      • 4 core and 5 core multicore formations are drawn as a single cable OD. Within cable core arrangement is not modelled.
      Open cable knowledge basis
      This interface is intended to expose geometry assumptions clearly and separate them from catalogue driven product selection.
      • British MV classes retained here are 6.35/11 kV and 19/33 kV.
      • Transmission class retained here is 64/110 kV IEC system.
      • Sector conductors are called out because they affect OD and terminations.
      • Stored ODs and bend data must still be verified before any formal design issue.
