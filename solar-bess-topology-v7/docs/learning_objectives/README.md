# V7 Learning Objectives

This folder records topics that must be understood before they become V7 upgrade packages.

The purpose is to prevent premature implementation of advanced electrical features before the assumptions, limits and engineering responsibilities are clear.

## Governing rule

```text
Learn first.
Define assumptions second.
Create training logic third.
Only then prepare an upgrade manifest.
```

## Current learning files

```text
01_advanced_electrical_layers_before_upgrade.md
```

Learning gate for future SLD, load flow, feeder loading, MCCB, ACB, protection and electrical study layers.

## Development rule

No advanced electrical feature should be installed into V7 until the relevant learning objective has been reviewed and converted into safe training language.
