# Single Line Diagram References

This page collects reference material for electrical single line diagrams used in the United Kingdom electricity system.

These examples come from transmission operators, distribution network operators and infrastructure systems. They provide real engineering diagram conventions that can be reverse engineered for grid modelling, documentation and automation.

---

## UK Power Networks

Distribution network symbol guide used in network schematics and system diagrams.

[UK Power Networks NetMAP Symbol Guide](https://assets.publishing.service.gov.uk/media/64db463cc8dee400127f1c5f/UKPN_2_Eastern_symbol_guide.pdf)

Regional development plan containing distribution network single line diagrams.

[UKPN Regional Development Plan Example](https://library.ukpowernetworks.co.uk/Library/GetPdf?pdfUrl=Asset_Management_Documents/Regional_Development_Plans/EPN/EPN_RDP_Elstree_Mill_Hill_Willesden_GSP.pdf)

## Notes

These references provide real examples of:

- transmission network diagrams
- distribution network topology
- substation configurations
- traction power supply systems
- electrical symbol conventions

They form a useful basis for understanding the graphical language used in electrical single line diagrams and can be used to reverse engineer diagram conventions for grid modelling and system documentation.
